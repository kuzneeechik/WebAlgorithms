Дерево решений

Это дерево решений.

Оно строится на основе твоих входных данных в формате csv текста и может

определять наиболее вероятный исход событий при заданных условиях.

Заполнять все условия необязательно.

Кроме того, ты можешь оптимизировать размер дерева, чтобы убрать критерии,

которые не играют важной роли.
